Dashboard
Centrální přehled celého systému
▶
Hlavní funkce
  • Přehledové karty — zobrazují počty záznamů z každého modulu (instalace, výroba, obhlídky, reklamace...)
  • Kalendář — barevně odlišené události ze všech modulů na jednom místě
  • Úkoly — rychlý task list s checkboxy pro denní úkoly
  • Zobrazit jako — přepínání pohledu za konkrétního uživatele (Ondra, David, Michal...)
Barvy v kalendáři
  • Instalace
  • Výroba
  • Obhlídky
  • Reklamace
  • Dovolená
Kliknutím na kartu modulu se dostanete přímo do daného modulu. Kliknutím na událost v kalendáři se otevře detail záznamu.

Instalace
Plánování a správa montáží
▶
Hlavní funkce
  • Tabulka instalací — přehled všech montáží s obj. číslem, místem, datem, technikem, stavem
  • Kalendář — vizuální zobrazení instalací po dnech, vícedenní zakázky se zobrazují jako pruhy
  • Partneři — databáze externích firem a řemeslníků s interaktivní mapou ČR
Stavy
  • Čeká na potvrzení
  • Naplánováno
  • Probíhá
  • Hotovo
  • Zrušeno
Jak přidat instalaci
  1. Klikněte na + Přidat instalaci
  2. Vyplňte obj. číslo, místo, obsah, datum a technika
  3. Vyberte stav a klikněte Uložit
Partneři — databáze řemeslníků
  • Klikněte na tlačítko 👷 Partneři v toolbaru
  • Otevře se sidebar s interaktivní mapou ČR — klikněte na kraj pro filtrování
  • Filtrujte podle oboru: Elektro, SDK, Malíř, Podlahy, Zedník, Klima
  • U každého partnera: hodnocení hvězdičkami, cena/hod, kontakt, poznámky
Kliknutím na hlavičku sloupce v tabulce seřadíte záznamy. Kliknutím znovu změníte směr řazení (↑/↓).

Výroba & Zakázky
Řízení výrobních zakázek
▶
Hlavní funkce
  • Tabulka zakázek — obj. číslo, zákazník, typ výroby, náročnost, deadline, technik, stav

  • Telegram notifikace — automatické upozornění na změny stavu
Stavy
  • Čeká na výrobu
  • Ve výrobě
  • Hotovo
  • Vydáno zákazníkovi
Nastavení (⚙)
  • Telegram Chat ID — zadejte ID pro příjem notifikací o změnách stavů zakázek
Urgentní zakázky se zobrazují v červeném pruhu nahoře. Číslo ve sloupci "Zbývá" se zbarví červeně, když je termín překročen.

Konfigurátor RING
Ceník a konfigurace RING svítidel
▶
Hlavní funkce
  • Kroková konfigurace — vybírejte parametry krok za krokem (velikost, barva, typ LED...)
  • Živý náhled — SVG vizualizace svítidla se mění podle výběru
  • Kalkulace ceny — cena se počítá v reálném čase podle zvoleného oboru
  • Správa cen — záložka pro editaci cen jednotlivých komponent
Jak konfigurovat
  1. Vyberte parametr v prvním kroku
  2. Pokračujte dalšími kroky — nedostupné kombinace jsou šedé
  3. Progress bar ukazuje kolik kroků zbývá
  4. Po dokončení se aktivuje tlačítko Kopírovat konfiguraci
Správa cen
  • Přepněte na záložku SPRÁVA CEN
  • Editujte ceny v Kč u jednotlivých komponent
  • Klikněte Uložit pro potvrzení nebo Zahodit pro návrat
Žlutý rámeček s vykřičníkem upozorňuje na nedostupné kombinace. Ceny se zobrazují přímo u každé volby.

Hlavní funkce

Lumibook PRO
Kniha svítidel — katalog a projekty
▶
Hlavní funkce
  • Projekt — nahoře vlevo: název projektu, zákazník, datum, číslo projektu, zpracoval
  • Import z eshopu — vložte URL produktu z ledshopik.cz a klikněte Načíst
  • Databáze produktů — sdílený katalog svítidel s parametry (výkon, lumen, CCT, CRI, IP)
  • Místnosti — organizace svítidel do místností (+ místnost pro vytvoření nové)
  • Karty svítidel — vizuální zobrazení s obrázkem, parametry, cenou
  • PDF export — export knihy svítidel do PDF
Jak přidat svítidlo
  1. V sekci Import z eshopu vložte URL produktu z ledshopik.cz
  2. Klikněte ⚡ Načíst — data se stáhnou automaticky
  3. Nebo klikněte + Nové svítidlo pro ruční zadání
  4. Přiřaďte svítidlo do místnosti a nastavte počet kusů
Klávesové zkratky
  • Ctrl + N — Nové svítidlo
  • Ctrl + P — Export PDF
  • Esc — Zavřít modální okno
Databáze produktů je sdílená — produkty importované na jednom zařízení jsou dostupné na všech. Přepínání mezi zobrazením Karty/Seznam je vpravo nahoře.

Kalkulátor instalací
Výpočet cen montáží a cestovného
▶
Hlavní funkce
  • Kalkulace vzdálenosti — zadejte PSČ odkud a kam, systém spočítá km
  • Položky instalace — přidávejte řádky s typem montáže, metry, cenou/m
  • Doplňkové práce — zapněte checkbox u přídavných prací (SDK, elektro...)
  • Celková kalkulace — automatický součet: práce + cestovné + doplňky
  • Historie — uložené kalkulace pro pozdější referenci
Nastavení sazeb (⚙)
  • Sazba za km — cestovní náklady
  • Hodinová sazba cestovní čas
  • Hodinová sazba montážník
  • Tabulka sazeb — ceny podle typu montáže, pásků, přípravy
  • Doplňkové práce — sazby za SDK, elektro atd.
Sazby se ukládají a platí pro všechny budoucí kalkulace. Historii kalkulací najdete pod hlavním formulářem.

Produktové listy
Generátor katalogových listů
▶
Hlavní funkce
  • Generátor — vložte URL produktu, systém stáhne data a vygeneruje list
  • Batch generování — vložte více URL najednou pro hromadné zpracování
  • Editace — upravte specifikace před generováním
  • Všechny listy — záložka s historií vygenerovaných listů
Jak vygenerovat list
  1. Vložte URL produktu z eshopu
  2. Klikněte Import — data se načtou automaticky
  3. Zkontrolujte a upravte specifikace
  4. Klikněte Generovat pro vytvoření listu
Pro hromadné generování použijte batch režim — vložte více URL oddělených novým řádkem.

Docházka & Absence
Evidence dovolených, nemocí a absencí
▶
Hlavní funkce
  • Kalendář — měsíční přehled absencí celého týmu s barevným rozlišením
  • Tabulka — seznam všech absencí s filtrováním a řazením
  • Automatický počet dnů — systém spočítá pracovní dny mezi daty
  • Export — stáhněte data jako CSV nebo PDF
  • Svátky — české státní svátky jsou automaticky zvýrazněny
Typy absencí
  • Dovolená — řádná dovolená
  • Neschopenka — pracovní neschopnost
  • Neplacené volno
  • Lékař — návštěva lékaře
Jak zadat absenci
  1. Klikněte + Přidat absenci
  2. Vyberte zaměstnance a typ absence
  3. Zadejte datum od–do — dny se spočítají automaticky
  4. Klikněte Uložit
Export dat
  • ↓ CSV — export do tabulky (Excel, Google Sheets)
  • ↓ PDF — export pro tisk
  • Filtrujte podle roku pomocí dropdownu Období
Svátky jsou automaticky zvýrazněny červeně v kalendáři. Víkendy se nepočítají do pracovních dnů. Kalendář zobrazuje absence celého týmu najednou.
